## Key Concepts

### Bonding Curve
```
Price increases as more people buy
Early buyers get cheaper price
Creates natural price discovery
```

### Fair Launch
```
No presale, no team allocation
Everyone buys at same curve
Prevents insider advantage
```

### Graduation
```
Token starts on launchpad (bonding curve)
At threshold (e.g., $69K market cap)
Migrates to real DEX (Raydium, Uniswap)
```
